# ADR-004: Official equivalence semantics

Status: Accepted — 2026-07-15

Only co-membership in an AIFA transparency-list `Codice gruppo equivalenza` is `AIFA_TRANSPARENCY_OFFICIAL`. Shortage declarations, same ingredient, same ATC and A/H prescription groups are separate evidence types. Public `official-equivalents` queries filter the official type in SQL and exclude the requested package itself.
